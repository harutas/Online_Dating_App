# online_dating_app

## URL
https://harutas.github.io/Online_Dating_App/

## Usage
* ユーザーカードまたはPROFILEボタンのクリックでユーザーの詳細を確認できます。
* MESSAGEボタンから個人チャット画面へ移動します。
* 右上の人型のアイコンからメッセージ一覧画面へ移動し、過去のメッセージの履歴を確認できます。
* Settingから性別を選択することで異性のみ検索することができます。  
その際、メッセージの履歴はリセットされるので注意してください。
